Entirely client-side. The source is location.hash and the sink is innerHTML. The server never sees your payload — it's after the #.

📜Vulnerable Code
// Runs on every hash change
window.addEventListener('hashchange', render);
function render() {
  const tab = decodeURIComponent(location.hash.slice(1));
  // BUG: hash value flows into innerHTML — server never sees the #fragment
  document.getElementById('tab-content').innerHTML = tab;
}

#1The # is not sent to server

The hash value flows into innerHTML. Choose a vector that executes via innerHTML — not <script> (it won't run), but onerror or SVG will.
