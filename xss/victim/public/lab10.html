window.name persists across cross-origin navigations. This page reads it and passes it to document.write() — letting the attacker set the source from their own origin.

📜Vulnerable Code
// Reads window.name set by the referrer page
const source = window.name || 'a friend';
// BUG: document.write with attacker-controlled source
